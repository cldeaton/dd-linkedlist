# linked-list-print


The package, allows the user to print a linked list as a String unit.

### Installing

```
npm install --save linked-list-print
```

And in Node:

```
var print = require('linked-list-print');

```

## Examples

```
[const print = require('linked-list-print');

const c = {
  value: 'c',
  next: undefined,
},

const b = {
  value: 'b',
  next: c,
}

const a = {
  value: 'a',
  next: b,
};


print(a);
// a->b->c
```
